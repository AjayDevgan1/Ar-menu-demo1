🚀 Firebase Setup Instructions:
1. Go to https://console.firebase.google.com/ and create a project.
2. Add a web app and get your Firebase config.
3. Replace the config object in firebase-config.js.
4. Create a Firestore database.
5. Add a collection named "orders".

📁 File Structure:
- index.html → Welcome screen
- ar.html → AR camera + model viewer
- chef.html → Dashboard for chef
- styles.css → All styles
- scripts.js → Logic for AR page
- firebase-config.js → Firebase setup
- /models/ → Put your pizza.glb and burger.glb files here.
- /assets/fish.png → Replace with your welcome image
